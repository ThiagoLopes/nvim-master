### Install
The process below will write/change the `.vimrc` file content.
```
$ git clone https://github.com/ThiagoLopes/vim-master.git ~/.vimconfigs
$ cd ~/.vimconfigs
$ sh install-vim.sh
$ vim -c ':PlugInstall'
```
